# Dailies del Proyecto

En esta carpeta se documentan las reuniones diarias del equipo.

El objetivo de las dailies es dejar constancia de:

- Que hizo cada persona el dia anterior.
- Que va a hacer cada persona durante el dia.
- Que bloqueos existen.
- Que decisiones se han tomado.
- Que Pull Requests o tareas necesitan revision.

## Formato recomendado

Crear un archivo por cada daily con este formato:

```text
YYYY-MM-DD-daily.md
```

Ejemplo:

```text
2026-06-17-daily.md
```

## Preguntas guia

Cada integrante deberia responder:

1. Que hice ayer?
2. Que voy a hacer hoy?
3. Tengo algun bloqueo?

## Buenas practicas

- Mantener las dailies breves.
- Documentar bloqueos de forma clara.
- Anadir enlaces a tareas de Jira cuando sea posible.
- Anadir enlaces a Pull Requests cuando sea necesario.
- Registrar decisiones importantes tomadas durante la reunion.
